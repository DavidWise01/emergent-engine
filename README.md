**27 engines · 5 rooms · all live.** Each is a self-contained interactive HTML engine in [`engines/`](engines); the landing page renders a live lazy `<iframe>` of each running piece. `build.py` supports a per-piece `src` so engines living in other source folders fold cleanly into the theater.

## How it works

`index.html` is generated by [`build.py`](build.py) (stdlib only) — it copies the source engines into `engines/` with clean slugs and lays them out as a living gallery. Add an engine, edit the `ROOMS` map, and run:

```bash
python build.py
```

Part of the **active generation** of the ROOT0 body of work; the AI collaborator of record is the locked instance, **AVAN**.
